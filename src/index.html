UI/UX Design Projects

Germany Travel Bureau

The goal of this project was to target new, existing and past visitors to travel back to Germany. To do so the present brand design wasn’t sharing any recognition or familiarity. To increase travel attraction and brand recognition sometimes simplicity is best. Adding features including themes from the Bauhaus and iconic colors from their flag and national soccer teams. I wanted a consistent design that screams Germany without question.

KC Firestone Events

The goal of this project was to provide a brand refresh for the user interface. The three target areas I chose to upgrade were the responsive website desktop view, the responsive website mobile view, and a Native iOS app. I completed an audit of the previous site where I pin-pointed the Heuristics and identified the changes that needed to be made. Next, I selected featured components that would bring value to the user. By hitting these targets I was able to bring a minimal and classy design straight to the user.

Monday.com

The goal of this project was to tell a story by solving inconveniences with the user experience on Monday.com. After completing research and reviewing interview transcripts, I was able to identify pain points, create a persona, and tell a story using a journey map, user flow, and site maps. Overall, I recommended the necessary changes by utilizing visual maps paired with research.
